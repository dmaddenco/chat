# chat
CS457 Chat Program using Sockets

To run the program as the chat client, use the following commands (with $ being terminal prompt):
$./chat -s <server IP address> -p <server PORT number>
$./chat -p <server PORT number> -s <server IP address>

To run the program as the chat server, use the following command (with $ being terminal prompt):
$./chat

To see the help message, use the following command (with $ being terminal prompt):
$./chat -h

Assumptions:
Only lowercase flags will be used to execute the program.
